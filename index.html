hexo发布相关操作

2020-07-19hexo

一、发布文章
1、进入hexo根目录，输入指令 hexo new “title”
2、根据创建成功的目录找到生成文件，进行编辑
3、文章设置目录

4、启动服务器，开始本地调试 hexo -s debug
5、引用图片
在全局配置文件（_config.yml)中将post_asset_folder设置为true；
新建文件时会在文件夹下同时生产title.md文件和titile文件夹
将图片放在title目录下，引入图片格式 ![description](title/pic_name)
mac截图按键（全屏 shift+alt+3 自定义 shift+alt+4）
6、发布到github上，回到根目录执行指令：
hexo clean && hexo g && hexo d
